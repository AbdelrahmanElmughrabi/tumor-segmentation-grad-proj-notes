# EXP02_01: Preprocessing Pipeline

## Goal
Establish a standardized MONAI preprocessing chain for mammograms.

## What Was Done
* Implemented lateral flips to face all breasts the same direction.
* Masked pectoral muscle in MLO views.
* Applied CLAHE for contrast enhancement and Z-score normalization.

## Outcomes
* ✅ Output tensors standardized to (1, 512, 512).
* ✅ Lesion visibility enhanced while masking distracting muscle.

## In Plain Words
We clean the images by hiding the chest muscle, adjusting the brightness so tumors pop out, and making sure all images face the same way.
