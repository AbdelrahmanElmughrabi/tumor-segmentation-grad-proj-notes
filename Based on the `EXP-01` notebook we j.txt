EXP-02 is essentially a image cleaning and preparation pipeline before the model sees any data.
Here's what each step does in plain terms:
Flip — some mammograms have the breast on the right side, others on the left. We flip the right ones so every image has the breast facing the same direction. We flip the mask too so it stays aligned.
Pectoral muscle removal — in MLO view mammograms there's a bright triangular muscle in the corner (the pectoral muscle) that has nothing to do with tumors. If we leave it in, the model might learn to associate that bright region with malignancy. We detect it and set it to black.
CLAHE — the raw 16-bit mammograms have subtle contrast differences that are hard to see. CLAHE is a contrast enhancement technique that makes the tumor region more visually distinct without over-brightening the whole image. Think of it like adjusting the local brightness in small 8×8 tiles across the image.
Z-score normalization — converts pixel values to a standard scale (mean=0, std=1) so the model doesn't get confused by raw pixel intensity differences between different scanners or patients. Only applied to the actual breast tissue, not the black background.
Output — a clean (1, 512, 512) float32 tensor ready to enter the EfficientNet-B0 encoder, with the ROI mask still perfectly aligned to it.
That's it — clean input in, model-ready tensor out.
