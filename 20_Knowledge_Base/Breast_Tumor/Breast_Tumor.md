## Context
Breast cancer is the most frequently diagnosed malignancy worldwide (~2.3M new cases in 2020). Deep learning approaches range from simple 2D classification to complex 3D volumetric segmentation pipelines. Our project targets the mammography modality with a combined segmentation + classification + explainability pipeline.

## Open Source Landscape Summary
The field has matured significantly from simple CNNs to foundation models. Key gap: no clean open source project combines segmentation + classification + Grad-CAM specifically for 2D mammography in a single pipeline.
